# 무료 크레딧 정책

## 신규 가입자 혜택

```json
{
  "new_user_benefits": {
    "registration_credit": 100,
    "valid_period": "30days",
    "usage_restrictions": {
      "daily_limit": 20,
      "excluded_features": ["premium_search"]
    }
  }
}
```

***

## 이벤트성 크레딧

### 1. 시즌별 이벤트

* 신년 이벤트: 100 크레딧
* 명절 이벤트: 50 크레딧
* 여름/겨울 특별 이벤트: 200 크레딧

### 2. 프로모션 크레딧

* 첫 매물 등록: 200 크레딧
* 친구 초대: 50 크레딧
* 연속 로그인: 최대 100 크레딧

***
